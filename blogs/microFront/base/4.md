# 创建 Parcel应用

## Parcel应用作用

+ Parcel用于创建公共UI, 涉及到跨框架共享UI时需要使用Parcel
+ Parcel的定义可以使用任何 single-spa支持的框架, 它也是单独的应用, 需要单独启动, 但是它不关联路由
+ Parcel应用的模块访问地址也需要被添加到 import-map 中, 其他微应用通过 System.import 方法进行引用

在这个parcel中, 放置链接, 通过点击链接, 可以在不同的微应用之间进行跳转
