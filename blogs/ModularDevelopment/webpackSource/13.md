# 编译主流程分析

```js
let webpack = require('webpack');

let options = require("./webpack.config");

// 在工作的时候需要接收的参数就是webpack的配置, 这个方法返回的结果也就是complier就可以进行编译了
let complier = webpack(options);

// 编译
complier.run((err, stats) => {
  // 错误优先
  console.info(err);
  console.info(stats.toJson());
});
```
